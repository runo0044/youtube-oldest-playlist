プライバシーポリシー

本プライバシーポリシーは、Chrome拡張機能「YouTube Oldest-First Playlist」（以下「本拡張」といいます）における、 ユーザーデータの取り扱いについて定めるものです。

1. 収集・保存する情報

本拡張は、機能の提供のために以下の情報をユーザーのブラウザ内に保存します。

  • APIキー
    ユーザーがGoogle Cloud Consoleから取得し、本拡張のUIで入力したYouTube Data API v3のAPIキー。
  • OAuth 2.0 クライアント情報
    ユーザーがGoogle Cloud Consoleから取得し、本拡張のUIでアップロードしたOAuth 2.0クライアントのJSONファイルの内容。

2. 一時的に処理する情報（保存しない情報）

以下の情報は、プレイリスト作成処理中にのみメモリ上で利用され、処理完了後に破棄されます。ブラウザ内に永続的に保存されることはありません。

  • Google認証トークン (OAuth Token)
    プレイリストの作成・編集のためにGoogleアカウントの認証（OAuth 2.0）を経て取得される一時的なアクセストークン。
  • チャンネル情報および動画IDリスト
    プレイリスト作成対象のチャンネルURLから取得した、チャンネルID、チャンネル名、およびそのチャンネルに属する動画のIDリスト。

3. 利用目的

収集・処理した情報は、以下の目的のみに利用します。

  • APIキー: YouTube Data APIを利用して、チャンネル情報（アップロード動画リストIDなど）や動画リストを取得するために使用します。
  • OAuth 2.0 クライアント情報: Googleの認証フローを開始し、ユーザーの操作を代理するための認証トークンを取得するために使用します。
  • 認証トークン: ユーザーに代わって、YouTube上に新しいプレイリストを作成し、そのプレイリストに動画を追加するために使用します。

これらの情報が上記以外の目的で利用されることは一切ありません。

4. 保存期間と保存場所

「1. 収集・保存する情報」で定義されたデータは、ユーザーのブラウザの同期ストレージ（`chrome.storage.sync`）にのみ保存されます。これにより、同じGoogleアカウントでログインしている複数のChromeブラウザ間で設定が同期されます。
これらのデータは、ユーザーが本拡張をアンインストールするか、ブラウザの拡張機能管理画面からデータ消去を実施すると、すべて完全に削除されます。

5. 第三者提供の有無

本拡張が保存・処理する情報が、GoogleのAPI（認証およびYouTube Data API）以外に送信されることはありません。開発者を含むいかなる第三者にも、これらの情報が提供・販売・共有されることは一切ありません。

6. セキュリティ対策

  • 全てのAPI呼び出しはHTTPSを用いて暗号化し、通信経路上の盗聴・改ざんを防止します。
  • 保存されたAPIキーやOAuthクライアント情報は、ChromeのストレージAPIによって保護されており、本拡張機能のスコープ外からアクセスすることはできません。

7. ユーザーの権利

ユーザーは、自身のデータについて、いつでも本拡張をアンインストールしたり、拡張機能の設定からデータを消去したりすることで、保存された情報を完全に削除する権利を有します。

8. プライバシーポリシーの変更

本プライバシーポリシーは、機能の変更や法令改正に応じて更新されることがあります。 更新があった場合は、本ドキュメントの最終更新日を更新します。重要な変更については、拡張機能の更新情報などを通じて通知します。
